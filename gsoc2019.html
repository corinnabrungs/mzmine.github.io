How to report issues

If you encounter a problem with MZmine then please file a new issue here. Please include the below information with your report.

  1. Your MZmine version
  2. Your OS version - is it 32-bit or 64-bit?
  3. The exact output of the following two commands:
    java -version
    R --version
  4. Screenshots of all the error messages or unexpected output
  5. The log messages reported in the MZmine console window
  6. The exact steps you took until you encountered the problem.
    Please list all the relevant parameter values you set for each data processing step. If you feel that the results provided by MZmine were wrong, please describe exactly why you think they were wrong and how they differ from the results you expected. It will help us a lot in tracking down the problem if you upload your MZmine project file to Dropbox or other storage system so we can reproduce your steps.
